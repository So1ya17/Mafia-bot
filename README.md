# Mafia Bot 🤖🎮

Telegram-бот для игры в **Мафию** с поддержкой **топиков**. Работает только в том топике, где был вызван, не мешая другим чатам.

## Стек

- **Python 3.10+**
- **aiogram 3** (async)
- **aiosqlite** (SQLite)
- **python-dotenv**

## Установка и запуск

```bash
git clone <repo>
cd mafia_bot
python -m venv .mafia && source .mafia/bin/activate
pip install -r requirements.txt
cp .env.example .env  # и вписать BOT_TOKEN
python bot.py
```

😈 **Мафия** — рядовой член

🎭 **Подставщик** — фальсифицирует проверку

🃏 **Серый кардинал** — цель: дожить до конца с нераскрытой ролью

## Фазы игры

```
Лобби (таймер) → Ночь → Обсуждение → Голосование → Линчевание → Ночь → ...
```

- **Лобби** — сбор игроков через deep-link кнопку, таймер продлевается при новых игроках
- **Ночь** — игроки с активными ролями выбирают цели в ЛС через inline-кнопки
- **Обсуждение** — живые игроки обсуждают в топике (мёртвые и зашатнажированные не могут писать)
- **Голосование** — все живые голосуют в ЛС, результаты публикуются в топик
- **Линчевание** — кнопки За/Против с live-счётчиком

## Экономика

## Особенности

- ✅ Полная поддержка **Telegram Topics** (работает только в вызванном топике)
- ✅ **Crazy Mode** — роли перемешиваются между живыми игроками каждый ход
- ✅ **Предсмертные сообщения** — убитые игроки могут оставить последние слова (сохраняются в БД)
- ✅ **Устойчивость к рестартам** — таймеры, игры и last words сохраняются в БД
- ✅ **Атомарный старт** — защита от двойного запуска
- ✅ **17 ролей** с уникальными механиками и балансом
- ✅ **Магазин предметов** — 13 предметов за монеты и алмазы
- ✅ **Once-per-game механика** — большинство предметов срабатывают только 1 раз за игру (таблица `game_item_usage`)
- ✅ **Алиби** — автоматический дополнительный голос при линчевании
- ✅ **Двойник** — перенаправляет ночное убийство на случайного живого
- ✅ **Отравленный кинжал + Антидот** — взаимодействующие предметы
- ✅ **Яд медленного действия** — отложенная месть через ночь
- ✅ **Карнавальная маска** — скрывает имя в списке живых на 2 ночи
- ✅ **Профиль и статистика** — `/profile` показывает игры, предметы, достижения; `/stats` — раскладку по ролям
- ✅ **Система достижений** — 16 ачивок с 3 уровнями (🥉🥈🥇) и наградами монетами
- ✅ **Передача предметов** — `/give @username item [количество]` в ЛС
